About

tokenPlease

by Smit Verma

tokenPlease is a browser extension that displays values from localStorage, sessionStorage, and cookies on matched pages — with one-click copy functionality. Perfect for debugging authentication tokens, session IDs, and API keys during development.

Features

📦
Multi-Source Support Read localStorage, sessionStorage, and cookies (including HttpOnly)
🌐
Flexible Domain Matching Match by hostname, full URL, wildcard (*.example.com), or suffix patterns
🔍
Regex Key Matching Use advanced regex patterns to match multiple keys at once, powering workflows
🔄
SPA-Safe Polling Automatic refresh with configurable interval, safe for single-page applications
🎭
JWT Decode One-click decode of JWT tokens with formatted payload preview, for ease of use
🛡️
Smart ID Automatically detect common auth keys like accessToken, PHPSESSID, next-auth tokens, and more
📋
One-Click Copy Copy any token value to clipboard with a single click, decreasing testing friction
🎨
Collapsible Overlay Non-intrusive floating UI that can be moved anywhere and minimized to a small circle

How to Use

1. Add a Rule

Go to the Rules tab and click Add Rule. Configure:

  • Name — A label for display in the overlay
  • Source — Choose localStorage, sessionStorage, or cookie
  • Domains — Enter domains to match (one per line)
  • Key — The storage key or cookie name to monitor

2. View Tokens

Visit any page matching your rule domains. The overlay appears at the top-right corner showing your configured tokens with:

  • Copy — Copy the raw value instantly
  • Mask/Reveal — Toggle between masked and visible values
  • JWT Decode — Decode and preview JWT payloads

3. Smart ID (Optional)

Enable Smart ID in the Behavior tab to automatically scan for common authentication keys. This feature:

  • Scans localStorage and sessionStorage for known auth key patterns
  • Works globally or per-rule when enabled in rule settings
  • Includes an option to include partial matches for broader detection

4. Import / Export

Backup and share your configuration using the Import / Export tab. Export your rules and settings as JSON, then import them on another browser or device.
